
# Math Game - C++ (Console Application)

This is a simple Math Quiz game built using C++ as part of my learning journey in Programming Advices (Course 5) with Dr. Mohammed Abu-Hadhoud.

## 🚀 Features
- 3 Difficulty Levels: Easy, Medium, and Hard.
- Mix Level: Randomly generates questions from all levels.
- Arithmetic Operations: Addition, Subtraction, Multiplication, and Division.
- Mix Operations: Randomly chooses the operation for each question.
- Final Results Summary: Shows the number of correct and wrong answers with color-coded feedback.

## 🛠️ Concepts Applied
- Structs: Used to manage game data and results.
- Enums: Used for Levels and Operation types.
- Pass by Reference: To efficiently update game results across functions.
- Functional Decomposition: Clean code structure with separated logic and UI functions.

## 📖 How to Run
1. Clone the repository.
2. Compile the main.cpp using any C++ compiler.
3. Run the executable and follow the on-screen instructions.
